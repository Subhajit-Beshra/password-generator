# password-generator

A simple, responsive web application to generate **secure, random passwords** with a clean UI and interactive features.

---

## Features

- Generate **strong random passwords** of customizable length.  
- Supports **uppercase, lowercase, numbers, and symbols**.  
- **Copy to clipboard** functionality for convenience.  
- Animated **gradient heading** and **container border** for a modern look.  
- Responsive design — works well on desktop and mobile.

---

---

## Technologies Used

- **HTML5** – Structure of the application.  
- **CSS3** – Styling, animations, and responsive layout.  
- **JavaScript** – Password generation logic and copy-to-clipboard functionality.
